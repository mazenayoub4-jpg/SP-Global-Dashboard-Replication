# S&P Global Market Intelligence Dashboard Replication

## Project Overview
I built a payment orchestration market dashboard from scratch, modeled directly after official S&P Global Market Intelligence financial reports. The goal was to take raw financial data, clean and model it properly, and present it in a high-end corporate layout that looks exactly like an industry publication.

## Technical Breakdown
* *Power BI & Data Modeling:* Cleaned the dataset and built the relationships between tables to ensure all slicers and filters cross-filter smoothly across the dashboard.
* *Advanced DAX:* Wrote custom measures to calculate Year-over-Year (YoY) growth, Compound Annual Growth Rate (CAGR), and dynamic revenue projections from 2024 to 2029.
* *UI/UX Design:* Built a clean, professional financial layout from scratch using a custom teal blue color palette to mimic top-tier corporate reporting.

## Core Features
* *Deep-Dive Filtering:* Added interactive slicers and page filters so users can easily segment and analyze specific market points.
* *Growth Forecasting:* Designed automated charts that visually map out complex revenue growth trends and percentage shifts over a 5-year outlook.
